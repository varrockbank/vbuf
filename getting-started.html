Getting Started

1. Required CSS

Include the css found in style.css for structural styles, then define your colors:

<link rel="stylesheet" href="style.css">
<style>
.wb {
  background-color: #282C34;
  color: #B2B2B2;
}
.wb .wb-selection {
  background-color: #EDAD10;
}
.wb .wb-cursor {
  background-color: #FF6B6B;
}
</style>

2. Required HTML Structure

All elements required. Missing any causes errors.

<blockquote class="wb no-select" tabindex="0" id="editor">
  <textarea class="wb-clipboard-bridge" aria-hidden="true"></textarea>
  <div class="wb-content">
    <div class="wb-gutter"></div>
    <div class="wb-lines"></div>
  </div>
  <div class="wb-status" style="display: flex; justify-content: space-between;">
    <div class="wb-status-left"><span class="wb-linecount"></span></div>
    <div class="wb-status-right">
      <span class="wb-coordinate"></span>
      <span class="wb-indentation"></span>
    </div>
  </div>
</blockquote>

Checklist:

3. Include the Script

Include the function Buffee found in Buffee.js

<script src="buffee.js"></script>

Or minified:

<script src="dist.buffee.min.js"></script>

4. Initialize

const editor = new Buffee(document.getElementById('editor'))
editor.Model.lines = ["Hello, World!"]
editor._internals.render(true);

5. Auto-fit Viewport

By default, the editor auto-fits to its container (if it has a defined height). To use fixed dimensions instead:

new Buffee(el, { viewportRows: 20 })      // Fixed row count
new Buffee(el, { viewportCols: 80 })      // Fixed column width
new Buffee(el, { viewportRows: 20, viewportCols: 80 }) // Both fixed

For auto-fit to work, editor (.wb class) must be: (a) 100% of parent or (2) set height.
