Gutter and Status Bar

Gutter visibility: showGutter option. Status bar: omit .wb-status element from template.

1. Default (gutter + status bar)

new Buffee(el)

Ln , Col |

2. No gutter

new Buffee(el, { showGutter: false })

Ln , Col |

3. No status line

Omit the .wb-status element from the template:

View template 
<div class="wb">
  <textarea class="wb-clipboard-bridge"></textarea>
  <div class="no-select wb-elements">
    <div class="wb-gutter"></div>
    <div class="wb-lines" tabindex="0">...</div>
  </div>
  <div class="wb-status">...</div>
</div>

4. Status line on top

Place .wb-status before .wb-elements in the template:

View template 
<div class="wb">
  <textarea class="wb-clipboard-bridge"></textarea>
  <div class="wb-status">...</div>
  <div class="no-select wb-elements">
    <div class="wb-gutter"></div>
    <div class="wb-lines" tabindex="0">...</div>
  </div>
</div>

Place .wb-gutter after .wb-lines in the template:
