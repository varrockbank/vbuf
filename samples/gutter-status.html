Gutter and Status Bar

Control visibility via constructor options: showGutter and showStatusLine

Default (gutter + status bar)

new Buffee(el, { showGutter: true, showStatusLine: true })

|

No gutter (line numbers hidden)

new Buffee(el, { showGutter: false })

|

No status bar

new Buffee(el, { showStatusLine: false })

|

Minimal (no gutter, no status bar)

new Buffee(el, { showGutter: false, showStatusLine: false })

|
