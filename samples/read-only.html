Read-Only Mode

Two options for read-only editors. Try typing - editing is disabled, but arrow keys work.

Option 1: TUI Mode (simpler)

JS:

new Vbuf(el, { initialViewportSize: 10 })

TUI.enabled = true

|

Option 2: Chunked Mode

JS:

new Vbuf(el, { initialViewportSize: 10 })

Model.activateChunkMode()

Model.appendLines([...])

|
