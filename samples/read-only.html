Read-Only Mode

Two options for read-only editors. Try typing - editing is disabled, but arrow keys work.

Option 1: TUI Mode (simpler)

new Buffee(el, { initialViewportSize: 10 })
TUI.enabled = true
|

Option 2: Chunked Mode (for large files)

BuffeeChunkLoader(editor)
ChunkLoader.activate()
await ChunkLoader.appendLines([...])
|
