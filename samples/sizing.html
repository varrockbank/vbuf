Editor Sizing with Character Units

The editor uses monospace font. Set width on the container element via CSS. Use ch units for predictable column counts, or 100% to fill parent.

Height (number of lines) is set via the initialViewportSize option in the constructor.

80 columns × 10 lines

CSS: width: 80ch;

|

1 line editor (no status)

JS:

new Vbuf(el, { initialViewportSize: 1, showStatusLine: false })

|

40 columns (no gutter)

CSS: width: 40ch;

JS:

new Vbuf(el, { showGutter: false })

|

40 columns × 5 lines

CSS: width: 40ch;

|

120 columns × 15 lines

CSS: width: 120ch;

|

100% width × 8 lines

CSS: width: 100%;
