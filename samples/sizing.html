Editor Sizing

The editor uses monospace font. Set width on the container element via CSS.

Use ch units for predictable column counts, or 100% to fill parent.

Height (number of lines) is set via the initialViewportSize option.

80 columns × 10 lines

CSS: width: 80ch;

|

1 line editor (no status)

new Buffee(el, { initialViewportSize: 1, showStatusLine: false })

|

40 columns (no gutter)

CSS: width: 40ch;

new Buffee(el, { showGutter: false })

|

40 columns × 5 lines

CSS: width: 40ch;

|

100% width × 8 lines

CSS: width: 100%;
